DOM -> Document Object Model (Modelo de Objetos para Documentos)

Definição: Conjunto de objetos dentro do navegador que dará acesso aos componentes internos dentro do website. Dessa forma, não funciona no Node JS.

Árvore DOM de um site: Mostra a hierarquia de todos os componentes e estruturas que formam uma página web. A raiz dessa hierarquia é o window. Dentro de window existem vários "filhos", colocados como os mais usados: o location, document e history.

  • Location: qual é a localização do site, qual é a página atual, pagina anterior.
  • Document: documento atual.
  • History: Paginas visitadas anteriormente.

Seleção dos elementos por:

  • MARCA: getElementsByTagName()
  • ID: getElementById()
  • NOME: getElementsByName()
  • CLASSE: getElementsByClassName()
  • SELETOR: querySelector() ou querySelectorAll()
